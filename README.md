# SI-TRUST 2.0 – Digitalizacija in GDPR nadgradnja

**Kaj rešujemo**  
Izdaja kvalificiranega digitalnega potrdila in aktivacija mobilne identitete brez obiska upravne enote.

**Tehnologije**  
Splet + mobilni tok, HSM/PKI, integracije (CRP, SI-PASS), varni API-ji, nadzorna plošča KPI/KRI.

**GDPR elementi**
- Pasica piškotkov + nastavitve privolitve (analitični privzeto izklopljeni)
- Izjava o zasebnosti (modal + povezava v footerju)
- ROPA/DPIA (kontrolni seznam, arhiv)
- DSAR izvoz osebnih podatkov (JSON/PDF)
- Politike hrambe in avtomatska anonimizacija/izbris
- Revizijska sled (append-only, žigi časa, IP, ID seje)

**KPI/KRI (COBIT MEA03)**
- Čas do izdaje ≤ 5 min
- Uspešne identifikacije ≥ 98 %
- Razpoložljivost ≥ 99,7 %
- Incidenti zasebnosti/varnosti = 0
